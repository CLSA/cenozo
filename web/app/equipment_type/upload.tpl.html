Upload {{ model.viewModel.record.name }} Equipment Data

This utility allows you to upload equipment data from a CSV file. The file must conform to one of the following options:

Option #1: three rows (used to upload new equipment)

  • Serial Number (must not belong to any other equipment type)
  • Site Name (may be blank)
  • Equipment Note (may be blank)

Option #2: eight rows (used to upload equipment that has already been loaned)

  • Serial Number (must not belong to any other equipment type)
  • Site Name (may be blank)
  • Equipment Note (may be blank)
  • UID (must be a pre-existing UID)
  • Lost (considered lost if value is 1, y, yes or true)
  • Start Date & Time in UTC (YYYY-MM-DD hh:mm format)
  • End Date & Time in UTC (YYYY-MM-DD hh:mm format, may be blank)
  • Loan Note (may be blank)
